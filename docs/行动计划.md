   ```<br>3. 编写 DbContext 并注册：<br>   ```csharp
   public class AppDbContext : DbContext

   builder.Services.AddDbContext<AppDbContext>(options =>
       options.UseNpgsql(builder.Configuration.GetConnectionString("NeonPg")));
   ``` | 1. 放弃 EF Code First，直接用 SQL 创建表（Neon 控制台执行更快捷）；<br>2. 只保留核心字段，放弃密码加密、外键级联等复杂配置 |
